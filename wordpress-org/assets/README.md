# Plugin Directory assets

Add only final, licence-cleared Plugin Directory artwork here before the
WordPress.org submission:

- `icon-128x128.png` and `icon-256x256.png`
- `banner-772x250.png` and `banner-1544x500.png`
- screenshots named `screenshot-1.png`, `screenshot-2.png`, and so on

These images are uploaded separately to the Plugin Directory SVN `/assets/`
path and must not be copied into the plugin release ZIP.
