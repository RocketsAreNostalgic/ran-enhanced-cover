# Third-party notices

RAN Enhanced Cover does not bundle third-party runtime libraries, fonts, images,
or remote services. The compiled block assets use WordPress-provided package
handles and browser platform APIs.

The repository uses WordPress build tooling and PHP quality tools during
development. Those development dependencies are not included in release ZIPs;
consult `package.json`, `pnpm-lock.yaml`, and `composer.lock` when present for
their complete dependency and licence metadata.
